About

이 페이지는 무엇인가요

arXiv에 매일 수백 건의 새 논문이 올라오지만, 그중 내 분야와 관련된 것을 골라 읽는 일은 점점 더 어려워지고 있습니다. 이 페이지는 컴퓨터 비전과 LLM에 집중해 읽을 만한 논문을 선별하고, 핵심을 한국어로 요약해 빠르게 파악할 수 있도록 돕기 위해 만들었습니다.

논문 데이터는 GitHub Actions로 매일 자동 수집되며(.github/workflows/daily-papers.yml), 요약은 GitHub Actions의 시크릿이 있으면 LLM으로, 없으면 규칙 기반 템플릿으로 생성됩니다 (요약 가이드라인). 브라우저에서 바로 열어 볼 수 있는 정적 페이지로, 별도의 서버나 API 키가 필요하지 않습니다.
